# autorestapi

Tested on Ubuntu 15.04 with NodeJS 4.2.1 installed earlier.

## Install

 1. `sudo ./seed --new arapi --owner $(whoami)`
 2. `arapi install`
 3. `arapi start`

## Notes

 1. `arapi server /ssl-new` Generates new SSL certificate. Prompts for host name.
 2. `arapi refresh` Restarts server.
 3. `arapi stop` Stops server.
